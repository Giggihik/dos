Клонировать репозиторий — это скопировать удалённый репозиторий на локальную машину. При клонировании на машину копируются файлы и папки проекта и вся его история.
Репозиторий Git — это место, в котором хранится весь код и вся история его изменений. По сути это просто папка, однако она связана с Git напрямую и содержит файлы в понятном для Git формате.
Запушить в Git — это отправить локальную ветку на удалённый репозиторий
